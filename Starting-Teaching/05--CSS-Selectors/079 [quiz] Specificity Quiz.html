    Question 3

    Given the following HTML:

    <section class="about">
  <h2 class="heading">Welcome To My Page</h2>
</section>

    And these styles:

    .about .heading {
  color: orange;
}

.about h2.heading {
  color: pink;
}

section.about .heading {
  color: yellow;
}

    What color will the h2 end up?

    1
    2
    3

    Correct Answer

    3

    yellow

    Explanation

    Correct! This selector is tied in specificity with the one above it, but it "wins" because it comes last in the file!
