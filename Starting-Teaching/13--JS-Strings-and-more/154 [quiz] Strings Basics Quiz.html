Question 1

Given the following string:

const animal = "hippopotamus"

Which character am I accessing?

animal[7]

1
2
3

Correct Answer

2

"t"

Explanation

Correct! "t" is the 8th letter in the word, but it has an index of 7 (JS starts counting at 0)

Question 2

What is stored in the combo variable?

const city = 'Kyoto';
const country = 'Japan';
const combo = city + country;

1
2
3

Correct Answer

2

"KyotoJapan"

Explanation

Correct! JavaScript doesn't add any spaces in, it just smashes two strings together!

Question 3

What does the following return?

let year = "1998";
year + 1;

1
2
3
4

Correct Answer

4

"19981"

Explanation

Correct! When adding a string and a number, JS behaves oddly! It turns the number into a string and smushes them together
